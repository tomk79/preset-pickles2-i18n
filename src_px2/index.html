Edit Sitemap

First, we will register the entire site structure in a sitemap, which is a CSV file where each line is represented by a page.

The sitemap CSV is located in the following path.

./px-files/sitemaps/

サイトマップを編集します

はじめに、サイト全体の構造をサイトマップに登録します。1行が1ページで表現されるCSV形式のファイルです。

サイトマップCSVは次のパスに置かれています。

./px-files/sitemaps/

Edit the theme

A theme is a PHP template that generates HTML source outside the content area.

It is located in the following path.

./px-files/themes/

テーマは、コンテンツエリア以外のHTMLソースを生成するPHPテンプレートです。

Edit the content

Content is a simple HTML file that contains only the source of the content area.
Create an HTML file in the path registered in the path field of the site map.

You can see the source of this page you are looking at by opening /index.html in a text editor.
Please delete or overwrite this page as well to replace it with the top page of your site.

コンテンツを編集します

コンテンツは、コンテンツエリアのソースだけが記述されたシンプルなHTMLファイルです。
サイトマップの path 欄に登録したパスにHTMLファイルを作成してください。

あなたがいま見ているこのページは、テキストエディタで /index.html を開くとソースを確認できます。
このページも、削除や上書きしてあなたのサイトのトップページに置き換えてください。

Publish

Once your site is complete, you can export it to static HTML as is.
Publish with the following command:

$ php ./.px_execute.php /?PX=publish.run

After a while, a static HTML file will be generated in the next path.

./px-files/dist/

パブリッシュします

サイトができあがったら、そのまま静的なHTMLに書き出すことができます。
次のコマンドで、パブリッシュを実行します。

$ php ./.px_execute.php /?PX=publish.run

しばらくすると、次のパスに、静的なHTMLファイルが生成されます。

./px-files/dist/
